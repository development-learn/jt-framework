# 协议扫盲

## 请先读我

`808`协议目前(写`jt-framework-2.0`文档的时刻即 `2021-12-15 01:45:17` ) 就 `2011`、`2013`、`2019` 三个版本。

::: danger

- 据官方文档描述 `2011` 和 `2013` 在消息整体结构上没任何区别(仅仅是消息体的区别)。

- `2019` 版在 **消息体属性** 和 **终端手机号** 之间新增了协议版本号属性。

:::

所以，本项目将 `V2011` 和 `V2013` 版统一处理(都叫 `V2013` )。
